EDUSparks Admin Dashboard - Functional Prototype

How to use:
1. Extract this ZIP.
2. Open admin.html in your browser.
3. Register a new admin using an email ending with @edusparks.my.
4. After registration, sign in using the same email and password.
5. Try Manage Collections, User Management, Messages/Feedback, Profile, and Settings.

Important:
- This is a frontend prototype. Data is saved in browser localStorage.
- It works like a real demo on the same browser, but it is not a secure production backend.
- To connect with the user website, keep this admin.html in the same folder as your EDUSparks index file.
